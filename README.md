# SNY TMS - Task Management System

Полнофункциональная система управления задачами и проектами.

## Функции
- ✅ Управление пользователями
- ✅ Управление задачами
- ✅ Управление проектами
- ✅ Удаление элементов

## Технологии
- Node.js
- Express.js
- MySQL
- HTML/CSS/JavaScript

## Установка
1. npm install
2. node server.js
3. Откройте http://localhost:3000
